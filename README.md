#  Автоматизация тест-кейсов для платформы модерации объявлений

Репозиторий содержит набор **E2E тестов**

**Стек технологий:** `Playwright` + `TypeScript` + `ESLint`.

---

## 📂 Структура проекта

```text
TC-automization-avito/
├── pages/                    # Page Object Model (Логика страниц)
│   ├── basePage.ts           # Базовый класс с общими методами
│   ├── mainPage/             # Главная страница и фильтры
│   └── statsPage/            # Страница статистики
├── tests/                    # Тестовые спецификации (.spec.ts)
│   ├── filtersTests/         # Тесты поиска и сортировки
│   ├── autoupdateStatsTests/ # Тесты таймера и обновления данных
│   └── themeTests/           # Тесты визуальных тем (Mobile)
├── data/                     # Константы и тестовые справочники
├── playwright.config.ts      # Глобальная конфигурация Playwright
├── package.json              # Скрипты запуска и зависимости
├── TESTCASES.md              # Описание сценариев (Manual)
└── BUGS.md                   # Найденные дефекты

```

🛠️## 🛠️ Предварительные требования

Перед запуском убедитесь, что в вашей системе установлены:

### 1. Node.js (v18.0+)

- **Проверка:** `node --version` (должно быть v18.x.x или выше)
- **Установка:** Скачать с [официального сайта](https://nodejs.org/). Рекомендуется LTS версия.

### 2. Git

- **Проверка:** `git --version`
- **Установка:** [Git SCM](https://git-scm.com/)

### 3. IDE (Рекомендуется)

- **Visual Studio Code** + расширение **Playwright Test for VSCode**

## 🚀 Быстрый старт

### Шаг 1: Клонирование репозитория

Откройте терминал и выполните:

```bash
#Клонирование репозитория
git clone https://github.com/doub1educk/QA-intern-Avito
# Переход в папку с клоном репозитория 
cd QA-intern-Avito
# Открытие проекта в IDE(ОПЦИОНАЛЬНО) 
code .
Далее если вы открыли проект в IDE, выполните Ctrl+shift+` для откртыие терминала прямо в IDE
``` 
> **Примечание:** Если Git не установлен, скачайте проект архивом (нажмите кнопку **Code** -> **Download ZIP**) и распакуйте его на вашем компьютере.

### Шаг 2: Установка окружения

Установите библиотеки и браузеры, необходимые для работы тестов. Выполните в терминале:

```bash
# Установка всех зависимостей проекта
npm install          

# Загрузка бинарных файлов браузеров Playwright
npx playwright install
```

### 🧪 Запуск тестов

В проекте настроены различные режимы выполнения тестов. Выберите подходящий под вашу задачу:

| Сценарий запуска | Команда |
| :--- | :--- |
| **Все тесты (Headless)** | `npx playwright test` |
| **Все тесты (Для просмотра прямой работы в браузере(не рекомендуется))** | `npx playwright test --headed` |
| **Интерактивный режим (UI)** | `npx playwright test --ui` |
| **Конкретный файл** | `npx playwright test tests/themeTests/themeSwitch.spec.ts` |
| **Тест по названию (-g)** | `npx playwright test -g "Нажатие «Обновить»"` |

### 🧹 Качество кода (Линтинг)

Для соблюдения единого стиля и чистоты кода в проекте используется **ESLint**.

**Команда для проверки всего проекта:**
```bash
npx eslint .
```
**Автоматическое исправление ошибок:**
```bash
npx eslint . --fix
```
**Отчеты о тестировании**
После каждого прогона генерируется подробный HTML-отчет. Чтобы открыть его, введите:
```bash
npx playwright show-report
```
