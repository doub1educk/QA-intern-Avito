### Тест-кейсы: Фильтрация, сортировка и мобильная верстка (Тёмная тема)

| ID | Название тест-кейса | Предусловие | Шаги воспроизведения | Ожидаемый результат |
|:---|:---|:---|:---|:---|
| **TC-1** | Применение фильтра “Диапазон цен” | В системе присутствуют объявления с ценами: 500, 1000, 5000, 10000 руб. | 1) Ввести в поле «От» значение: 1000<br>2) Ввести в поле «До» значение: 5000<br>3) Нажать Enter для подтверждения | На экране отображаются только объявления, удовлетворяющие диапазону (>= 1000 и <= 5000) |
| **TC-2** | Сортировка “По цене” (возрастание) | В системе присутствуют 2 и более объявления с разными ценами | 1) Нажать на список “Сортировать по”<br>2) Выбрать пункт “Цене”<br>3) Нажать на список “Порядок”<br>4) Выбрать “По возрастанию”<br>5) Нажать Enter<br>6) Проверить набор объявлений | Каталог объявлений отсортирован от меньшей цены к большей |
| **TC-3** | Фильтрация по категории «Недвижимость» | Наличие минимум 2 объявлений разных категорий (одно из них — “Недвижимость”) | 1) Нажать на выпадающее список “Категория”<br>2) Выбрать пункт “Недвижимость” | На экране отображаются только объявления с категорией “Недвижимость” |
| **TC-4** | Работа тогла “Только срочные” | В каталоге минимум 2 объявления (одно с пометкой “Срочно”) | 1) Перевести тогл “Только срочные” в активное состояние<br>2) Проверить результат выдачи | Отображаются только объявления “Срочно”. Фильтр “Приоритет” сменил значение на “Срочный” |
| **TC-5** | Кнопка для ручного обновления статистики | **Перейти на страницу “Статистика”** | 1) Нажать кнопку “Обновить” | Таймер автообновления обнуляется, данные статистики актуализируются |
| **TC-6** | Кнопка “Отключить автообновление” | **Перейти на страницу “Статистика”** | 1) Нажать на кнопку остановки автообновлений | Данные не обновляются сами. Вместо кнопки «Пауза» появилась кнопка «Включить» |
| **TC-7** | Кнопка “Включить автообновление” | **Перейти на страницу “Статистика”** | 1) Нажать кнопку остановки автообновлений<br>2) Нажать кнопку запуска автообновлений | Статистика обновляется по таймеру. Появилась кнопка остановки, таймер возобновил работу |
| **TC-8** | Переход на темную тему (Mobile) | Светлая тема активна. Открыта мобильная версия (devtools/эмулятор) | 1) Нажать на переключатель цветовой темы | Фон становится тёмным, текст и элементы интерфейса — светлыми |
| **TC-9** | Переход на светлую тему (Mobile) | Темная тема активна. Открыта мобильная версия (devtools/эмулятор) | 1) Нажать на переключатель цветовой темы | Фон становится светлым, текст и элементы интерфейса — тёмными |
